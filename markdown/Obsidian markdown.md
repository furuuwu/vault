## Lists

### unordered

use a dash/hyphen (`-`) and a space. Alternatives: asterisk (`*`), plus sign `+`

this is a list:
- amazing
- insane man
- crazzzyyyy

this is another one:
* super
* hyper
* awesome

+ hi
+ my
+ name
+ is

- today
- i
- wrote this

### ordered

1. this is the first element
2. second
3. third
4. fourth
5. ...


### nesting

- amazing
- insane man
	- nested
	- also nested
- crazzzyyyy


1. first
	1. one
	2. two
		1. one
		2. two
2. second
3. third

### todo lists

it's a list but renders as a checklist on preview mode. Each item has a "checkbox" `[ ]` that you can fill with any letter to mark as completed

- [ ] eat
- [ ] sleep
- [ ] pray
